Hey, I'm Wahib!

I've been making games since high school — the moment I realized I could make the games I play was world-changing, and I never stopped. I'm now three years into a Software Engineering degree at Umm Al-Qura University in Saudi Arabia, and every free hour goes back into building — polished systems, better tools, and games that feel good to play.

I love making games feel good and polished while keeping everything organized through modular architecture. Whether I'm building a tool or designing a system, my focus is always on making hard things easier — for myself and for my teammates.

I head the Game Development Committee at UQU's Electronic Games Club, where I lead and mentor over 150 students in Unity and C#. We've shipped 11+ games to itch.io together. I run workshops, plan sprints, coordinate multi-team releases, and watch people discover the same feeling I had the first time something they wrote moved on screen.

003

EASY DIALOGUE

Unity Editor Tool
Open Source

Every dialogue system I found was either too heavyweight or made writers feel like they were filing a tax return. I built Easy Dialogue as the tool I wished existed — a clean graph-based workflow, a custom EditorWindow with visual node connections, built-in validation, and a runtime layer that any Unity project can drop in with minimal setup.

C# Unity Editor API EditorWindow CustomInspector ScriptableObjects Open Source
Custom Graph EditorWindow
Nodes are draggable, connectable ScriptableObject assets on disk — the graph IS the data, not a separate representation of it.
Zero-Dependency Runtime
Single MonoBehaviour, no third-party dependencies. Reads ScriptableObjects directly, evaluates flag conditions, fires events. Drop in and go.

004

PATHNPC

Unity Editor Tool
Open Source

A visual, waypoint-based NPC pathing system for Unity. Click-to-place waypoints directly in the Scene View with real-time preview — no code needed to set up patrol routes. Supports multiple named paths per NPC, ping-pong and loop modes, per-waypoint pause durations, and color-coded path visualization.

C# Unity Editor API CustomInspector NavMesh Scene View Handles Open Source
Visual Scene View Editing
Click-to-place waypoints directly in the Scene View with real-time path preview, Gizmo handles, and full undo support. Zero friction for designers.
Multi-Path Runtime Switching
Multiple named paths per NPC with runtime switching. Inspector dashboard shows path statistics — total distance, waypoint count, estimated patrol time.
