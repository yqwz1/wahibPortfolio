PORTRAIT IMAGES — drop your photos here
========================================

Files expected by the site:

  about-me.jpg         — Your personal photo (About Me section)
  maybeteam.jpg        — MaybeTeam / Finn game portrait (Experience row 1)
  uqu-club.jpg         — UQU Electronic Games Club portrait (Experience row 2)
  freelance.jpg        — Freelance / tools work portrait (Experience row 3)

Recommended size: ~400×400 px minimum, square crop preferred.
Format: JPG or PNG (update the src attribute in index.html if using PNG).

Until you drop in a real image the site shows a "AFFIX PHOTO HERE" stencil placeholder.
